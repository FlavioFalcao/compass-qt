    Compass Example

    Compass is a Forum Nokia example application that teaches the use of a traditional compass and allows the user to determine the bearing to the desired location using Ovi maps. The main purpose of the example application is to demonstrate the use of the Maps and Navigation API. The example is hosted at Projects Forum Nokia: https://projects.forum.nokia.com/compass

    Required capabilities

    Application can be self-signed.

    Compatibility
